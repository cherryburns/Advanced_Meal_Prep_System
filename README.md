# Advanced_Meal_Prep_System
Interactive Python system for meal prep scheduling with resource management and task dependencies.
A Python simulation that manages meal preparation by assigning resources (stoves, ovens, air fryers, chefs), handling dependencies between dishes, and scheduling tasks using topological sorting.
Features:
Add dishes (Starter, Main Course, Dessert)

Automatic resource allocation

Define dependencies between dishes

Task scheduling with topological sort

Interactive menu-driven interface
